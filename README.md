Consultas SQL avanzadas, JOINs y gestion de datos en Azure SQL

**# Proyecto de Análisis SQL en Azure 📊

## Descripción del Proyecto
Este proyecto contiene una serie de ejercicios prácticos realizados sobre una base de datos *Azure SQL* utilizando el conjunto de datos *SalesLT*. Se enfoca en la manipulación de datos, creación de consultas complejas, uniones de tablas (JOINs) y gestión de registros (DML).

## Herramientas Utilizadas 🛠️
* *Base de Datos:* Azure SQL Database.
* *Lenguaje:* T-SQL (Transact-SQL).
* *Editor:* Azure Query Editor & Visual Studio Code.

## Contenido de las Consultas 📂
El script ejercicios_joins.sql incluye:
* *Selección y Filtrado:* Ordenación por precio y filtros avanzados de productos.
* *Análisis de Negocio:* Uso de COUNT, SUM, GROUP BY y TOP para identificar tendencias de ventas.
* *INNER JOINs:* Relaciones complejas entre clientes, direcciones y categorías de productos.
* *Gestión de Datos:* Inserción de registros en tablas personalizadas y manejo de valores NULL.
